# Amazon Clone

A front-end clone of Amazon's homepage built using HTML and CSS. This project was created to practice web development fundamentals, including page structure, layouts, styling, and responsive design.

## Features

* Amazon-inspired homepage design
* Navigation bar with search section
* Hero/banner section
* Product category cards
* Footer section
* Clean and organized layout

## Technologies Used

- HTML5
- CSS3
- Font Awesome (for icons)

## Project Structure

Amazon-Clone/

├── index.html
├── style.css
├── image-files
└── README.md

## Getting Started

1. Clone this repository:

   git clone <repository-url>
  
2. Open the project folder.

3. Open `index.html` in your web browser.
